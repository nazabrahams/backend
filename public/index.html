Path Method Description
Users Table
/users/ GET Gets all users
/users/:id GET Gets a single user by ID
/users/ POST Adds a new user
/users/:id DELETE Deletes a user by ID
/users/:id PATCH Updates user information by ID
Bookings Table
/bookings/ GET Gets all bookings
/bookings/:id GET Gets a single booking by ID
/bookings/ POST Adds a new booking
/bookings/:id DELETE Deletes a booking by ID
/bookings/:id PATCH Updates booking information by ID
/cart/ POST Adds a booking to the cart
